Detailed Description

Specialised for propagating to.

The top level of the lhs of assignment-like statements (including arguments in calls) is not modified. So for example eax := ebx -> eax := local2, but in m[xxx] := rhs, the rhs and xxx are modified, but not the m[xxx]

Note
This class' visitor functions don't return anything. Maybe we'll need return values at a later stage.
