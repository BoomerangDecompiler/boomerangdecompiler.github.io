Detailed Description

template<typename T, typename Sorter = void, typename Enabler = std::enable_if<std::is_base_of<Exp, T>::value>>
class ExpSet< T, Sorter, Enabler >

A class ordered or unordered sets of expressions.

Template Parameters
Tthe type of expression to store in the set
SorterBinary functor type that defines the sorting order. If Sorter == void, the set is unordered.
