72  /**
73  * Create a new Basic Block for this CFG.
74  * If the BB is blocked by a larger complete BB, the existing BB will be split at the first
75  * address of \p bbInsns; in this case this function returns nullptr (since no BB was created).
76  * The case of the new BB being blocked by a smaller complete BB is not handled by this method;
77  * use \ref ProcCFG::ensureBBExists instead.
78  *
79  * The new BB might also be blocked by exising incomplete BBs.
80  * If this is the case, the new BB will be split at all blocking incomplete BBs,
81  * and fallthrough edges will be added between parts of the split BB(s).
82  * In this case, the incomplete BBs will be removed (since we just completed them).
83  *
84  * \param bbType Type of the new Basic Block
85  * \param bbInsns list of instructions contained in this BB. Must not be empty.
86  *
87  * \returns the newly created BB, or the exisitng BB if the new BB is the same as
88  * another exising complete BB.
89  */
90  BasicBlock *createBB(BBType bbType, const std::vector<MachineInstruction> &bbInsns);
91  BasicBlock *createBB(BBType bbType, const std::list<MachineInstruction> &bbInsns);

100  /**

176  /**
177  * Split \p bb into a "low" and "high" part at the RTL associated with \p splitAddr.
178  * The type of the "low" BB becomes fall-through. The type of the "high" part becomes the type
179  * of \p bb.
180  *
181  * \ | / \ | /
182  * +---+ bb +---+ BB1
183  * | | +---+
184  * | | ==> | Fallthrough
185  * +---+ +---+
186  * / | \ +---+ BB2
187  * / | \
188  *
189  * If \p splitAddr is not in the range [bb->getLowAddr, bb->getHiAddr], the split fails.
190  * \param bb pointer to the BB to be split
191  * \param splitAddr address of RTL to become the start of the new BB
192  * \param newBB if non zero, it remains as the "bottom" part of the BB, and splitBB only
193  * modifies the top part to not overlap. If this is the case, the RTLs of the original BB
194  * are deleted.
195  *
196  * \returns If the merge is successful, returns the "high" part of the split BB.
197  * Otherwise, returns the original BB.
198  */
