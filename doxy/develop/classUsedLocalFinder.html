Detailed Description

This class differs from the above in these ways: 1) it counts locals implicitly referred to with (cast to pointer)(sp-K) 2) it does not recurse inside the memof (thus finding the stack pointer as a local) 3) only used after fromSSA, so no RefExps to visit.
