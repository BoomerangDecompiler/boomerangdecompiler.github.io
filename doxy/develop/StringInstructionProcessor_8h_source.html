40  * frag -> +----+ +----+ <= frag

45  * +----+ V

51  * | S' | | | S' = S less the skip and repeat parts
52  * +-b2-+ | | \p rptBranch is a branch for the repeat part

54  * V /
55  * +----+ < newFrag
56  * | B |
57  * | |
58  * +----+
