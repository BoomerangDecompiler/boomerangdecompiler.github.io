Detailed Description

Convert any exp{-} (with null definition) so that the definition points instead to an implicit assignment (exp{0}) Note it is important to process refs in a depth first manner, so that e.g.

m[sp{-}-8]{-} -> m[sp{0}-8]{-} first, so that there is never an implicit definition for m[sp{-}-8], only ever for m[sp{0}-8]
