Detailed Description

Transform an exp by applying mappings to the subscripts.

This used to be done by many Exp::fromSSAForm() functions. Note that mappings have to be done depth first, so e.g. m[r28{0}-8]{22} -> m[esp-8]{22} first, otherwise there wil be a second implicit definition for m[esp{0}-8] (original should be b[esp+8] by now)
