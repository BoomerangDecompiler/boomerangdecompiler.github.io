Detailed Description

template<typename T, typename Sorter = void, typename Enabler = std::enable_if<std::is_base_of<Statement, T>::value>>
class StmtSet< T, Sorter, Enabler >

This class implements an ordered or unordered set of statements.

Template Parameters
TStatement type (Statement, Assign, CallStatement etc.)
SorterBinary functor type that defines the sorting order. If Sorter == void, the set is unordered.
