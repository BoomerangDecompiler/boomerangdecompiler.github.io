30 /**
31  * Class for managing an interface plugin.
32  *
33  * - The plugin must define the following functions (with extern "C" linkage):
34  * - IFC* initPlugin(): to initialize the plugin class and allocate resources etc.
35  * You must ensure the returned pointer is valid until deinitPlugin() is called.
36  * - void deinitPlugin(): to deinitialize the plugin and free resources.
37  * - const PluginInfo* getInfo(): To get information about the plugin.
38  * May be called before initPlugin().
39  */
