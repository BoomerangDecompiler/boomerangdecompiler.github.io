28 /**
29  * StmtModifier is a class that for all expressions in this statement, makes a modification.
30  * The modification is as a result of an ExpModifier; there is a pointer to such an ExpModifier in a
31  * StmtModifier. Even the top level of the LHS of assignments are changed. This is useful e.g. when
32  * modifiying locations to locals as a result of converting from SSA form, e.g. eax := ebx -> local1
33  * := local2 Classes that derive from StmtModifier inherit the code (in the accept member functions)
34  * to modify all the expressions in the various types of statement. Because there is nothing
35  * specialised about a StmtModifier, it is not an abstract class (can be instantiated).
36  *
37  * \note This class' visitor functions don't return anything. Maybe we'll need return values at a
38  * later stage.
39  */
