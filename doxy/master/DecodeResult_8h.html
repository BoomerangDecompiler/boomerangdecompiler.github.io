class __attribute__ ( (visibility("default"))  )

The DecodeResult struct contains all the information that results from calling the decoder.

This prevents excessive use of confusing reference parameters.

Resets all the fields to their default values.

< Indicates whether or not a valid instruction was decoded.

The class of the instruction decoded. Will be one of the classes described in "A Transformational Approach to Binary Translation of Delayed Branches". Ignored by machines with no delay slots.

If true, don't add numBytes and decode there; instead, re-decode the current instruction. Needed for instructions like the Pentium BSF/BSR, which emit branches (so numBytes needs to be carefully set for the fall through out edge after the branch)

< The number of bytes decoded in the main instruction

The RTL constructed (if any).

If non zero, this field represents a new native address to be used as the out-edge for this instruction's BB. At present, only used for the SPARC call/add caller prologue
