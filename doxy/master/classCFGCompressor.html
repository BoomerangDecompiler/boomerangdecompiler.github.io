bool CFGCompressor::compressCFG ( ProcCFG *  cfg)

Given a well-formed ProcCFG, optimizations are performed on the graph to reduce the number of basic blocks and edges.

Assumes that cfg is well-formed.

Optimizations performed are:

  • Removal of redundant jumps (e.g. remove J in A->J->B if J only contains a jump)
  • Removal of empty BBs (not containing any semantics), if possible
  • Removal of BBs not reachable from the entry BB.
See also
ProcCFG::isWellFormed
Returns
true if the ProcCFG was changed.
