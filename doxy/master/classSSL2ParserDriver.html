SharedExp SSL2ParserDriver::makeSuccessor ( SharedExp  e)

Make the successor of the given expression, e.g.

given r[2], return succ( r[2] ) (using opSuccessor). We can't do the successor operation here, because the parameters are not yet instantiated (still of the form param(rd)). Actual successor done in Exp::fixSuccessor()

Note
The given expression should be of the form r[const]
The parameter expresion is copied (not cloned) in the result

The modified expression
