Detailed Description

Class for managing an interface plugin.

  • The plugin must define the following functions (with extern "C" linkage):
    • IFC* initPlugin(): to initialize the plugin class and allocate resources etc. You must ensure the returned pointer is valid until deinitPlugin() is called.
    • void deinitPlugin(): to deinitialize the plugin and free resources.
    • const PluginInfo* getInfo(): To get information about the plugin. May be called before initPlugin().
