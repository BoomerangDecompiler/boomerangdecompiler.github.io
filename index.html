Boomerang
Code Bugs Download Wiki

A free, open source machine code decompiler

Overview

Boomerang is a BSD licensed machine code decompiler written in C++. It runs on Linux, macOS and Windows. For the full list of supported ISAs and file formats see the main repository page.

Technical documentation

Documentation of the source code exists for the master branch and for the develop branch. The content is updated whenever the content of the main repository changes.
